# Solana Drip Go

Solana Drip Go Clients generated via dcaf-mocha/anchor-go

## Getting Started

install

```bash
go get github.com/dcaf-labs/solana-drip-go
```

## Adding a New Version

```bash
./generate.sh <version> <path to idl>
# ./generate.sh 0 ./idl.json
```
